## INSTALLATION ##
1. upload the package from "UPLOAD" to your MyBB root directory
2. add CSS from STYLES.txt to your board stylesheets (e.g. global.css)
3. add {$dvz_shoutbox} to your index template
4. install the plugin in the ACP
